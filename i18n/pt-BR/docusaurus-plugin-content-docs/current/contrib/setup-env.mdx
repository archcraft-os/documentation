# Configuração do Sistema

Vamos configurar seu sistema para desenvolvimento local...

- Instalar `nodejs`, `npm` e `yarn` no seu sistema :
```bash
$ sudo pacman -S nodejs npm yarn
```

- CLone o repositório `documentation` do Github :
```bash
$ git clone https://github.com/archcraft-os/documentation
```

- Instalação necessária do `node modules` no diretório raiz do repositório :
```bash
$ cd documentation
$ npm install
```

- Execute o servidor de desenvolvimento local​ :
```bash
$ yarn run start
```

- Adicione documentações, edite páginas e, quando terminar, faça o Build do site :
```bash
$ yarn run build
```
:::note

Certifique-se de que o build do site foi criado com sucesso e teste a compilação no servidor local antes de implantá-lo no Github (ou crie um pull request).

:::
