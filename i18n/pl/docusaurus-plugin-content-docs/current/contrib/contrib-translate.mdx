# Przetumacz Wiki

Postpuj zgodnie z tym poradnikiem, jeli chcesz przetumaczy Archcraft Wiki...

## Wymagania wstpne

- [Skonfigurowany system](setup-env.mdx) do pracy lokalnej.
- Powiniene mie podstawow wiedz na temat skadni [Markdowna](https://www.markdownguide.org/cheat-sheet) i [MDXa](https://mdxjs.com). Zobacz [Docusaurus Markdown Features](https://docusaurus.io/docs/markdown-features).
- Powiniene przynajmniej wiedzie jak: *zforkowa*, *zklonowa* repozytorium i utworzy *pull requesta*.

## Konfiguracja

Edytuj plik `docusaurus.config.js` i dodaj odpowiednie **[i18n locale](https://www.npmjs.com/package/i18n-iso-countries)**.

Na przykad: Jeli chcesz tumaczy na jzyk *rosyjski*, dodaj <kbd>ru</kbd> w sekcji `locales:` :
```jsx {4-8}
  // Tumaczenie
  i18n: {
    defaultLocale: 'en',
    locales: ['en', 'fr', 'ru'],
      ru: {
        label: 'Russian',
        direction: 'ltr',
      },
  },
```
