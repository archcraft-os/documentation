## Introdução

Essa edição especal do Archcraft vem com [`Sway`](/docs/wayland-compositors/sway), [`Wayfire`](/docs/wayland-compositors/wayfire), [`River`](/docs/wayland-compositors/river), [`Hyprland`](/docs/wayland-compositors/hyprland) e [`Newm`](/docs/wayland-compositors/newm) pré instalados, pré configurados e traz a você uma experiência única no Linux.

### Instalação

Para criar um USB inicializável, veja [Criar um USB Botável](/docs/boot-iso/boot-with-usb) <br/>
Para instalação gráfica, veja [Instalar o Archcraft com Calamares](/docs/install-archcraft/install-with-calamares) <br/>
Para instruções de instalação CLI, veja [Instalar o Archcraft com ABIF](/docs/install-archcraft/install-with-abif)

### Atualização

Se você já é um usuário WSE, você não precisa reinstalar a ISO para atualizar o sistema. Siga os passos abaixo para atualizar seu WSE:

- Na base regular, apenas execute o comando seguinte para atualizar o seu sistema -
```bash
$ sudo pacman -Syyu
``` 
- Quando uma nova ISO do WSE for lançada, um **Pacote de Atualização** també é laçado para usuários WSE existentes. Use este pacote para atualizar coisas específicas no WSE.
  - Baixe o pacote de atualização `existing-wse-update-MONTH-YEAR.tar.gz` no ko-fi.
  - Extraia com: `tar -xzvf existing-wse-update-MONTH-YEAR.tar.gz`
  - Vá para o diretório extraído: `cd existing-wse-update-MONTH-YEAR`
  - Abra o terminal nesse diretório e execute o script de atualização: `./01_update.sh`
  - É isso.

## Compatibilidade

### Máquinas Virtuais

Essa ISO funciona em VMs (Virtualbox, VMware, Qemu, etc). Mas **somente os Window Managers** (X11) vão funcionar. Todos os **compositores wayland não funcionam**. Se funcionar, serão muito lentos devido ao suporte 3D precário.

### Nvidia

Não existe suporte oficial para Nvidia. Infelizmente, seus drivers são muito confusos, e seu produtos aleatórios, é impossível ajudar se os `copositores wayland nessa ISO` não funcionar na sua máquina. todo caso pode ser aleatório, e funcionar perfeitamente, ou não funcionar de jeito nenhum. Então...<br\>
A `ISO` tem tudo configurado **fora-da-caixa**, para rodar um _compositor wayland_ usando **Nvidia**. É preciso de todos os módulos habilitados no `mkinitcpio.conf` e no **initrd**, `parâmetros do Kernel` adicionados, e é necessário os drivers de vídeo proprietários instalados.<br\>
***Agora, é sua escolha tentar isso.***

**Atualização:** Como reportado, usuários são capazes de inicializar e instalar a ISO nas suas máquinas Nvidia.

**Mais informações:** [Instalação de Sway](https://wiki.archlinux.org/title/Sway#Installation), [Hyprland Nvidia](https://wiki.hyprland.org/Nvidia/), [Requerimentos do wayland](https://wiki.archlinux.org/title/wayland#Requirements), [DRM_kernel_mode_setting](https://wiki.archlinux.org/title/NVIDIA#DRM_kernel_mode_setting), [Instalação#NVIDIA](https://wiki.archlinux.org/title/NVIDIA#Installation)

### Intel Iris XE

Alguns usuários reportaram que os Compositores Wayland dessa ISO não funcionam em sistemas com GPUs `Intel Iris XE`.

**Atualização: ** Eu pessoalmente testei a ISO com um sistema com `Nvidia + Intel iris XE` system (***Dell Inspiron 5520***) e a ISO funcionou bem.

## FAQs

<details>
<summary><b>Como usar Archcraft WSE?</b></summary>

Para qualquer ajuda com as Configurações e Atalhos de Teclado, visite a página do window manager e compositos wayland respectivo.

</details>

<details>
<summary><b>Como atualizar a ISO/Pacote de atualização?</b></summary>

Siga os passos abaixo para atualizar (ou baixar os arquivos novamente):
- Faça login na sua conta do Ko-fi
- Vá para seu `Histórico de pagamentos Ko-fi`
- Selecione suas compras
- Click em `Ver Detalhes`
- depois, clique em `Ver Conteúdo` para baixar os arquivos atualizados.

</details>

<details>
<summary><b>Posso ter o reembolso se a ISO não funcionar para mim?</b></summary>

Primeiro de tudo, este é o **modelo de doação** do Archcraft, onde você `doa para o suporte` do seu projeto open-source favorito.

O Archcraft não aceita doações. Essas coisas exclusivas ajudam o Archcraft em todas as suas necessidades financeiras. A ideia por trás disso, **Você ajuda o Archcraft e em troca você ganha um presente. Ganhar-ganhar para ambos!**

Todos os cenários são claramente mencionados nesta página([veja a compatibilidade](/docs/premium/wse#compatibility)), sobre o funcionamento desta ISO. Se você ***ainda escolher continuar com isso*** (sabendo todos os detalhes sobre isso) e a ISO não funcionar para você. **`Por favor, nao pergunte sobre reembolso ou trocas`**. Eu vpu, definitivamente, te ajudar com seu problemas e tentar resolvê-los o mais rápido possível.

</details>

<details>
<summary><b>Eu preciso de ajuda com a ISO</b></summary>

Os melhores lugares para se encontrar ajuda rápida são nas **Comunidades Archcraft**. Então junte-se à elas e procure por ajuda lá. Nos links no rodapé da página

Se você tiver problemas com isso, mande-me um e-mail para **adi1090x@gmail.com** e sinta-se à vontade para pedir ajuda.

</details>
