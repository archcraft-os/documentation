**Ubuntucraft : Experiência do Archcraft no Ubuntu**

[![video](https://raw.githubusercontent.com/archcraft-os/misc-files/main/archcraftify/ubuntucraft/main.gif)](https://youtu.be/nTnqDM8qNTk)

## Visão geral

- Transforme seu Ubuntu no **Archcraft**
- Window manager `openbox` totalmente configurado
- Window manager `bspwm` totalmente configurado
- Window manager `i3wm` totalm4nte configurado **`Novo`**
- Muitos **temas/estilos** para ambos (openbox e bspwm) WMs
- Tudo mais que o Archcraft oferece

:::warning
**Ubuntucraft** não é uma ISO ou uma distribuição baseada no Ubuntu
:::

**Ubuntucraft** é um pacote de instalação para transformar seu [Ubuntu](https://ubuntu.com/) no Archcraft. Uma vez instalado em cima do Ubuntu, ele irá ter tudo que o Archcraft oferece no seu Ubuntu.

**Importante** : Ubuntucraft é criado em cima do `Ubuntu 22.04 LTS Desktop` e é ***recomendado*** que seja instalado em uma nova instalação do Ubuntu 22.04 LTS.

:::info
Ele pode funcionar em uma versão mais antiga ou mais nova do Ubuntu, só não foi testado em outras versões.
:::

### Installação

Siga os passos abaixo para instalar o Ubuntucraft:

- Primeiro, **Instale o Ubuntu 22.04 LTS** (`Instalação Mínima`) no seu computador
- Baixe o `ubuntucraft.tar.gz` pelo ko-fi
- Transfira ou salve o arquivo na sua instalação do Ubuntu
- Extraia o arquivo: `tar -xzvf ubuntucraft.tar.gz`
- Mude para o diretório extraído: `cd ubuntucraft`
- Execute o script de instalação e deixe-o fazer o trabalho: `./install.sh`
- Depois de finalizado _reinicie seu computador_ e você terá o **Ubuntucraft instalado**.

:::info[Video]
Veja o [vídeo de instalação](https://youtu.be/nTnqDM8qNTk)
:::

:::tip[Asciinema]
O arquivo **asciinema** `ubuntucraft.cast` é fornecido, você pode executá-lo para ver como o script trabalha antes de usá-lo.
:::
