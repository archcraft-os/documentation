**Voidcraft : Experiência do Archcraft no Void Linux**

[![video](https://raw.githubusercontent.com/archcraft-os/misc-files/main/archcraftify/voidcraft/main.gif)](https://youtu.be/dd0bCn5Mt2E)

## Visão geral

- Transforme seu Void Linux no **Archcraft**
- Window manager `openbox` totalmente configurado
- Window manager `bspwm` totalmente configurado
- Window manager `i3wm` totalm4nte configurado **`Novo`**
- Muitos **temas/estilos** para ambos (openbox e bspwm) WMs
- Tudo mais que o Archcraft oferece

:::warning
**Voidcraft** não é uma ISO ou uma distribuição baseada no Void Linux
:::

**Voidcraft** é um pacote de instalação para transformar seu [Void Linux](https://voidlinux.org/) no Archcraft. Uma vez instalado em cima do Void Linux, ele irá ter tudo que o Archcraft oferece no seu Void Linux.

**Importante** : Voidcraft é criado em cima do `Void Linux Live Base (glibc)`(_void-live-x86_64-20240314-base.iso_) e é ***recomendado*** que seja instalado em uma nova instalação do Void Linux.

### Installation

Siga os passos abaixo para instalar o Ubuntucraft:

- Primeiro, **Instale o Void Linux** no seu computador
- Baixe o `voidcraft.tar.gz` pelo ko-fi
- Transfira ou salve o arquivo na sua instalação do Ubuntu
- Extraia o arquivo: `tar -xzvf voidcraft.tar.gz`
- Mude para o diretório extraído: `cd voidcraft`
- Execute o script de instalação e deixe-o fazer o trabalho: `./install.sh`
- Depois de finalizado _reinicie seu computador_ e você terá o **Voidcraft instalado**.

:::info[Video]
Veja o [vídeo de instalação](https://youtu.be/dd0bCn5Mt2E)
:::

:::tip[Asciinema]
O arquivo **asciinema** `ubuntucraft.cast` é fornecido, você pode executá-lo para ver como o script trabalha antes de usá-lo.
:::
