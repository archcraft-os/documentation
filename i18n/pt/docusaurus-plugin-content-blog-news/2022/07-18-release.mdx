Lançamento de julho corrigiu vários preblemas, pacotes atualizados, etc.

### Changelogs :
- A **base ISO** foi atualizada de acordo com `archiso 65-1` (grub para boot UEFI)
- Usando compressão `zstd` para mkinitcpio
- Corrigido o `consolefont` e outros avisos na geração do init
- Atualizados os arquivos **sddm config** e convenção de nomenclatura
- Atualizados os arquivos relacionados a usuários e grupos
- Corrigido o problema do `openbox` como sessão padrão no sddm 
- Atualizados todos os scripts e arquivos de configuração para todos os WMs
- **`Novo BSPWM`** : Recriados a configuraçãobase, scripts e **Nova Theme Engine (Infinite Themes)** 
- Adicionado o **Modo Fácil** no `openbox`, com ícones grandes, fontes e elementos UI
- Etc
