### Changelogs :
- ISO criada do zero
- Grub2 bootloader com temas
- Plymouth & SDDM customizados
- Todo XFCE4 Desktop com Xfce goodies
- Somente aplicações básicas, sem programas desnecessários
- Instalador Calamares (Gráfico)
- Suporte completo para Network Manager, Muitos plugins de VPN
- Suporte para Bluetooth integrado
- Pipewire para Som/Áudio (Systemwide, Bluetooth, Jack)
- Suporte para impressora integrado
- Quase todos os codes de ádio, vídeo e imagem
- Total funcionalidade do gestor de arquivos (Montagem, Networking, Arquivamento, etc)
- Suporte para o AUR integrado
- Interface do usuário mínima
- Muitos estilos de Desktop
- Ícones, temas, Wallpapers, fontes do Archcraft
- Etc
