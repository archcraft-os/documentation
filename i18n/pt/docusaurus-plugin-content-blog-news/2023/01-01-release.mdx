**`Principal Release`**

> Você precisa reinstalar o Archcraft para essa atualização. Depois disso, você pode atualizar para a próxima versão com `pacman`

### Changelogs :
- ISO criada do zero
- Suporte para Network Manager melhorado, vários plugins para VPN
- Suporte para Bluetooth melhorado
- Pipewire para Som/Áudio (Systemwide, Bluetooth, Jack)
- Suporte para impressora melhorado
- Quase todos os codecs de áudio, vídeo e imagem
- Funcionalidade do gestor de arquivos melhorada(Montagem, Networking, Arquivamento, etc)
- Suporte para multi monitor melhorado
- Fontes CJK pre instaladas
- Novos temas para openbox e bspwm
- Criação de temas pelo usuário é muito fácil agora
- Mais algumas pequenas mudanças, etc.
