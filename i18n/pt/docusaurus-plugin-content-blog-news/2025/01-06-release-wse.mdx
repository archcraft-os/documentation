<Highlight color="#e06b74">Exclusivo</Highlight> A nova **ISO Especial Archcraft Wayland Edition** já está disponível para download. Leia os registros de alterações abaixo para saber o que há de novo.
<!--truncate-->

Esta edição especial do Archcraft vem com [`Sway`](/docs/wayland-compositors/sway), [`Wayfire`](/docs/wayland-compositors/wayfire), [`River`](/docs/wayland-compositors/river), [`Hyprland`](/docs/wayland-compositors/hyprland) e [`Newm`](/docs/wayland-compositors/newm) pré-instalados, pré-configurados e oferece uma experiência Linux exclusiva para Wayland.
### Changelogs:  
- Perfil ISO atualizado para `archiso 82-1`  
- Base mais recente com novos pacotes  
- Atualização de todos os pacotes Archcraft e AUR  
- `Hyprland` atualizado para a versão mais recente  
- Correção do compositor `newm` quebrado  
- Diversas correções de bugs e muito mais!  

> O ***pacote de atualização de janeiro*** para sua **instalação WSE existente** já está disponível para download.  
>  
> Siga [estas instruções](https://wiki.archcraft.io/docs/premium/wse#update) para realizar a atualização.  

**Download:** Você pode baixar a ISO mais recente em [**`ko-fi`** ☕](https://ko-fi.com/s/213becbf00)

**Obtenha a ISO Atualizada:** Siga os passos abaixo para obter os arquivos atualizados (ou rebaixar os arquivos) gratuitamente:  
- Faça login na sua conta Ko-fi  
- Vá até o seu `Histórico de Pagamentos no Ko-fi`  
- Selecione as compras que você realizou  
- Clique em `Ver Detalhes`  
- Em seguida, clique em `Ver Conteúdo` para baixar os arquivos atualizados.
