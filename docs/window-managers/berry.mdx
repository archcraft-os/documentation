|Keybinding|Function|
|--|--|
|**mod+Return** | Opens a terminal (alacritty) |
|**mod+shift+Return** | Opens a terminal in floating mode |
|**mod+t** | Opens a terminal in fullscreen mode |
|**alt+F1/mod** | Opens app launcher |
|**mod+n** | Opens network menu |
|**mod+r** | Runs apps as root |
|**mod+m** | Opens the music menu |
|**mod+s** | Opens screenshoting menu |
|**mod+x** | Opens a powermenu |
|**mod+f** | Opens file manager (thunar) |
|**mod+w** | Opens a web browser (firefox) |
|**mod+e** | Opens a text editor GUI (geany) |
|**alt+ctrl+v** | Opens a TUI text editor (vim) |
|**alt+ctrl+r** | Opens a TUI file manager (ranger) |
|**alt+ctrl+h** | Opens a TUI system ressources monitor (HTOP) |
|**alt+ctrl+l** | Launches the berry lockscreen |
|**Print** | Takes a screenshot |
|**alt+Print** | Takes a screenshot (5s delay) |
|**Shift+Print** | Takes a screenshot (10s delay)|
|**ctrl+Print** | Takes a screenshot of active window |
|**ctrl+alt+Print** | Takes a screenshot of an area |
|**mod+Escape** | Reload keybindings|
|**mod+q/super+shift+c** | Kill an App |
|**mod+shift+q** | Quit berry |
|**mod+shift+Left/Right/Up/Down** | Resize the window |
|**mod+ctrl+Left/Right/Up/Down** | Move the window |
|**mod+Left/Right** | Snap the window |
|**mod+Up** | Set window to monocle |
|**mod+Down** | Set window to floating |
|**mod+space** | Set window to fullscreen |
|**mod+{1-9}** | Switch to another workspace |
|**mod+Shift+{1-9}** | Send focused window to another workspace |
|**mod+t** | Toggle window titlebar |
|**mod+Tab** | Cycle windows |
|**mod+c** | Center window |
